# 收工交通消息

這是一個關於收工時間交通消息notification的chrome extension。(只限於香港)

當你設定了收工時間之後，在星期一至五每日收工前的一小時內，這個工具就會把最新的交通消息更新推送給你。

另外，我們在顯示頁面也加入了Google map live traffic及運輸署全港主要道路交通情況的link讓你可以快速地看到現在的香港最新的路面交通情況。
